# GBase 图标说明

## 需要添加的文件
文件名：`icon_gbase.png`
位置：`frontend/src/assets/datasource/icon_gbase.png`

## 图标要求
- 格式：PNG
- 尺寸：建议 64x64 或 128x128 像素
- 透明背景
- 设计：GBase 官方 Logo 或数据库图标

## 临时解决方案
在获取正式 GBase 图标之前，可以：
1. 使用 GBase 官方网站提供的 Logo
2. 创建简单的文字图标（蓝色背景 + "GB" 白色文字）
3. 复制 MySQL 图标作为临时占位符

## 添加步骤
1. 准备 GBase 图标文件（icon_gbase.png）
2. 将图标放置到 `frontend/src/assets/datasource/` 目录
3. 图标会自动被前端代码引用（已在 ds-type.ts 中配置）

## 图标来源建议
- GBase 官方网站
- 设计师定制
- 在线图标生成工具
