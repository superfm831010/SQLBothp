# 黄埔海关定制化修改日志

## 2025-10-07: 修改欢迎语为黄埔海关版本

### 修改内容

将问数页面的欢迎语从 "SQLBot" 改为 "黄埔海关智能问数助手"。

### 修改文件

**文件：** `frontend/src/i18n/zh-CN.json`

**具体修改：**

1. **第184行 - 主欢迎语**
   - 原文：`"greeting": "你好，我是SQLBot，很高兴为你服务"`
   - 修改为：`"greeting": "你好，我是黄埔海关智能问数助手，很高兴为你服务"`

2. **第521行和第590行 - 自我介绍文本**
   - 原文：`"i_am_sqlbot": "你好，我是SQLBot"`
   - 修改为：`"i_am_sqlbot": "你好，我是黄埔海关智能问数助手"`

### 影响范围

- 聊天界面的初始欢迎消息
- 嵌入式智能客服的自我介绍
- 所有使用这些国际化字段的前端组件

### 后续步骤

如需在前端查看效果，请重新构建前端：
```bash
cd frontend
npm run build
```

或在开发模式下查看：
```bash
cd frontend
npm run dev
```

### 备注

- 仅修改了简体中文版本（zh-CN.json）
- 英文版（en.json）和韩文版（ko-KR.json）未修改，仍保持 "SQLBot" 名称
- 如需修改其他语言版本，请相应修改对应的国际化文件
